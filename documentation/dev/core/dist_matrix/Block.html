BlockMatrixÂ¶

The BlockMatrix<scalarType> is the parent class of each partial specialization DistMatrix<scalarTypee,colDist,rowDist,BLOCK> of DistMatrix to block-wise distributions; it is a child class of AbstractDistMatrix<scalarType>. Each of the derivatives of ElementalMatrix<scalarType> chooses a sensical pairing of distributions for the rows and columns of the matrix.
