Special cases used in Elemental¶

This list of special cases is here to help clarify the notation used throughout Elemental’s source (as well as this documentation). These are all special cases of DistMultiVec<T>.

template<>
class DistMultiVec<Real>¶

Used to denote that the underlying datatype Real is real.

template<>
class DistMultiVec<Complex<Real>>¶

Used to denote that the underlying datatype Complex<Real> is complex with base type Real.

template<>
class DistMultiVec<F>¶

Used to denote that the underlying datatype F is a field.

template<>
class DistMultiVec<Int>¶

When the underlying datatype is a signed integer.
