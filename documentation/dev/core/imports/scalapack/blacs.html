BLACS¶

A handful of Basic Linear Algebra Communication Subprograms (BLACS) routines are currently exposed to Elemental if MATH_LIBS was detected to contain a recent version of ScaLAPACK.

type blacs::Desc¶

Currently a typedef to std::array<int,9>.

While the following routine is not a wrapper around a BLACS routine, it is the most common means of filling a blacs::Desc from a BlockMatrix:

blacs::Desc FillDesc(const BlockMatrix<F> &A, int context)¶

Form the descriptor for the distributed matrix A using the specified context.
