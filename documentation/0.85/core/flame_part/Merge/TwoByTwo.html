2x2 matrix of blocks¶

The following routines merge a two-by-two set of contiguous submatrices which have conformal leading dimensions, sizes, and, if applicable, alignments. That is to say, if the inputs are consistent, a view of \(\begin{pmatrix} B_{TL} & B_{TR} \\ B_{BL} & B_{BR} \end{pmatrix}\) is returned.

Note

The constraints described above for horizontally and vertically merging submatrices both apply here.
